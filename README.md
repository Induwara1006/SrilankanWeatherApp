# Sri Lanka App

A Flutter mobile application for Sri Lanka.

## Getting Started

This is a Flutter project that supports multiple platforms including Android, iOS, Web, Windows, Linux, and macOS.

### Prerequisites

- Flutter SDK (version 3.32.2 or higher)
- Dart SDK
- For Android development: Android Studio and Android SDK
- For iOS development: Xcode (macOS only)
- For Windows development: Visual Studio with "Desktop development with C++" workload

### Installation

1. Clone or open this project in VS Code
2. Run `flutter pub get` to install dependencies
3. Run `flutter doctor` to check your development environment setup

### Running the App

To run the app on your desired platform:

```bash
# Run on connected device or emulator
flutter run

# Run on specific platform
flutter run -d chrome        # For web
flutter run -d windows       # For Windows
flutter run -d android       # For Android
flutter run -d ios          # For iOS
```

### Project Structure

- `lib/` - Main application code
- `test/` - Unit and widget tests
- `android/` - Android-specific code
- `ios/` - iOS-specific code
- `web/` - Web-specific code
- `windows/` - Windows-specific code
- `linux/` - Linux-specific code
- `macos/` - macOS-specific code
